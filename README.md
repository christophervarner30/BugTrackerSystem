# Bug Tracker System

A bug tracking application that allows users to report, track, and manage software bugs.

## Features
- Track bugs with detailed information
- Priority Level system (Low, Medium, High)
- Color-coded display of priorities
- Unit tests to verify functionality

## Technology Stack
- ASP.NET Core MVC
- .NET 9.0
- C#
- Bootstrap for UI components

## Development
This project demonstrates implementing feature branches and pull requests for safe code integration.
